# 功能说明：其他功能

## 功能概述

微笑/刷牙目标次数设置、首页任务列表布局优化、登录后日历初始化、应用卡片样式优化。

---

## 功能列表

### 1. 微笑/刷牙目标次数

微笑和刷牙任务支持设置 1-50 次的目标次数，之前被硬编码排除。

### 2. 首页任务列表布局

首页改为顶部日历固定、下方任务列表可滚动的结构，滚动时日历始终可见。

### 3. 登录后日历初始化

登录后进入主页时自动加载当月打卡日历数据，确保日历正确显示已打卡日期。

### 4. 应用卡片样式优化

任务卡片（AgencyCard）和进度卡片（ProgressCard）外观与交互全面升级。

---

## 核心业务代码

### 首页布局（HomeComponent.ets 关键结构）

```typescript
// 顶部日历固定，下方任务列表可滚动
Column() {
  HomeTopView({ homeStore: this.homeStore })
  List({ space: 12 }) {
    ForEach(this.homeStore.taskList, (item: TaskInfo) => {
      ListItem() {
        TaskCard({ taskInfo: item, homeStore: this.homeStore })
      }
    })
  }
  .layoutWeight(1)
}
```

### 登录后日历初始化（MainPage.ets）

```typescript
onPageShow() {
  let isForeground = this.isForeground;
  this.isForeground = true;
  if (isForeground || this.homeStore.dateArr.length === 0) {
    this.homeStore.onAction();
  }
}
```

if (this.showExpPopup) {
