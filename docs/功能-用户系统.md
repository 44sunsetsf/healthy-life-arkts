# 功能说明：用户系统

## 功能概述

完整的用户系统，包括账号登录、登录持久化、退出登录、账号与昵称分离、多用户资料隔离、个人资料展示和编辑、个性签名、头像上传。

---

## 功能列表

### 1. 账号登录

用户使用账号+密码登录应用。账号仅允许字母、数字和下划线，至少3位。预置6个示例账号（默认密码均为 123123）。

### 2. 登录持久化

登录成功后自动保存状态，下次打开应用跳过登录页直接进入主页。

### 3. 退出登录

"我的"页面底部红色按钮，点击后清除登录状态跳转登录页。

### 4. 账号昵称分离

账号（username）用于登录验证，昵称（nickname）用于界面显示，互不影响。

### 5. 多用户资料隔离

不同账号拥有独立的个人资料和打卡数据，切换账号后自动刷新。

### 6. 个人资料展示和编辑

支持编辑头像、昵称、签名、性别、生日、身高、体重，点击保存写入数据库。

### 7. 个性签名

新增签名栏，空值时显示占位文字"暂无个性签名"。

### 8. 头像上传

从手机文件系统选择图片（jpg/png/gif/bmp/webp）作为头像。

## 核心业务代码

### 登录逻辑（LoginPage.ets）

```typescript
handleLogin() {
  if (!this.username || !this.password) {
    prompt.showToast({ message: '请输入账号和密码' });
    return;
  }
  if (!this.isValidUsername(this.username)) {
    prompt.showToast({ message: '账号只能包含字母、数字和下划线' });
    return;
  }
  FriendInfoApi.login(this.username, this.password, (result: FriendInfo | null) => {
    if (result) {
      GlobalContext.getContext().setObject('currentUser', result);
      this.saveLoginState(result.username);
      router.replaceUrl({ url: 'pages/MainPage' });
    } else {
      prompt.showToast({ message: '账号或密码错误' });
    }
  });
}

saveLoginState(username: string) {
  let preferences = data_preferences.getPreferences(this.context, H_STORE);
  preferences.then((res) => {
    res.put(IS_LOGIN, true).then(() => {
      res.put(LOGIN_USER, username);
      res.flush();
    });
  });
}
```

### 登录状态判断（AdvertisingPage.ets）

```typescript
goToNextPage() {
  let preferences = data_preferences.getPreferences(this.context, H_STORE);
  preferences.then((res) => {
    res.get(IS_LOGIN, false).then((isLoginVal: data_preferences.ValueType) => {
      let isLogin = isLoginVal as boolean;
      if (isLogin === true) {
        res.get(LOGIN_USER, '').then((usernameVal: data_preferences.ValueType) => {
          let username = usernameVal as string;
          FriendInfoApi.queryByUsername(username, (user: FriendInfo | null) => {
            if (user) {
              GlobalContext.getContext().setObject('currentUser', user);
            }
            router.replaceUrl({ url: 'pages/MainPage' });
          });
        });
      } else {
        router.replaceUrl({ url: 'pages/LoginPage' });
      }
    });
  });
}
```

### 退出登录（MinePage.ets）

```typescript
handleLogout() {
  let preferences = data_preferences.getPreferences(this.context, H_STORE);
  preferences.then((res) => {
    res.put(IS_LOGIN, false).then(() => {
      res.flush();
      router.replaceUrl({ url: 'pages/LoginPage' });
    });
  });
}
```

### 加载当前用户资料（MinePage.ets）

```typescript
loadUserInfo() {
  let currentUser = GlobalContext.getContext().getObject('currentUser') as FriendInfo;
  if (currentUser && currentUser.username) {
    UserInfoApi.queryByUsername(currentUser.username, (result: UserInfo | null) => {
      if (result) {
        this.nickname = result.nickname || currentUser.nickname;
        this.signature = result.signature || '';
        this.avatar = result.avatar;
      } else {
        this.nickname = currentUser.nickname;
      }
    });
  }
}
```

### 头像上传（ProfilePage.ets）

```typescript
pickAvatar() {
  let documentSelectOptions = new picker.DocumentSelectOptions();
  documentSelectOptions.maxSelectNumber = 1;
  documentSelectOptions.fileSuffixFilters = ['jpg', 'jpeg', 'png', 'gif', 'bmp', 'webp'];
  let documentViewPicker = new picker.DocumentViewPicker();
  documentViewPicker.select(documentSelectOptions).then((documentSelectResult: Array<string>) => {
    if (documentSelectResult && documentSelectResult.length > 0) {
      this.avatar = documentSelectResult[0];
    }
  });
}
```

### 保存个人资料（ProfilePage.ets）

```typescript
saveUserInfo() {
  let currentUser = GlobalContext.getContext().getObject('currentUser') as FriendInfo;
  let username = currentUser ? currentUser.username : '';
  let userInfo = new UserInfo(0, username, this.avatar, this.nickname,
    this.signature, this.gender, this.birthday, this.userHeight, this.userWeight);
  UserInfoApi.insertOrUpdateData(userInfo, (flag: number) => {
    if (flag) {
      prompt.showToast({ message: '保存成功' });
    } else {
      prompt.showToast({ message: '保存失败' });
    }
  });
}
```
