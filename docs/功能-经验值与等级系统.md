# 功能说明：经验值与等级系统

## 功能概述

打卡获得经验值，累积升级。包含经验值计算、经验值动画、进度条、打卡经验弹窗、升级弹窗。

---

## 功能列表

### 1. 经验值系统

每次打卡获得 20 经验值，100 经验值升一级。等级和经验值显示在"我的"页面。

### 2. 经验值动画

打卡后切换到"我的"页面时，经验值数字从旧值递增到新值播放动画。

### 3. 经验值进度条

以进度条展示当前经验值占升级所需经验值的比例，格式如 20/100。

### 4. 打卡经验弹窗（ExpGainPopup）

打卡后弹出橙色胶囊提示"+20 EXP"，数字递增，1.8秒后自动消失。

### 5. 升级弹窗（LevelUpPopup）

升级时弹出金色光晕缩放动画弹窗，显示新等级，半透明遮罩背景。

## 核心业务代码

### 打卡+经验值逻辑（HomeViewModel.ets）

```typescript
static readonly EXP_PER_CLOCK: number = 20;
static readonly EXP_PER_LEVEL: number = 100;

public async taskClock(taskInfo: TaskInfo) {
  let taskItem = await this.updateTask(taskInfo);
  let expGained: number = 0;
  let oldLevel: number = 0;
  let newLevel: number = 0;
  if (taskItem.isDone) {
    let expResult = await this.addExperience(HomeStore.EXP_PER_CLOCK);
    expGained = expResult.expGained;
    oldLevel = expResult.oldLevel;
    newLevel = expResult.newLevel;
    let dayInfo = await this.updateDayInfo();
    if (dayInfo && dayInfo?.finTaskNum === dayInfo?.targetTaskNum) {
      achievementLevel = await this.updateAchievement(this.selectedDayInfo.dayInfo);
    }
  }
  return {
    achievementLevel, showAchievement, expGained, oldLevel, newLevel,
    isLevelUp: newLevel > oldLevel
  } as AchievementInfo;
}
```

### 经验值增加+同步排行榜（HomeViewModel.ets）

```typescript
addExperience(exp: number): Promise<ExpResult> {
  return new Promise((resolve, reject) => {
    GlobalInfoApi.query((res: GlobalInfo) => {
      let oldExp = res.experience;
      let oldLevel = HomeStore.getLevel(oldExp);
      res.experience += exp;
      GlobalInfoApi.updateData(res, (flag: number) => {
        if (flag) {
          let newLevel = HomeStore.getLevel(res.experience);
          AppStorage.SetOrCreate('experienceOld', oldExp);
          AppStorage.SetOrCreate('experienceNew', res.experience);
          AppStorage.SetOrCreate('experienceCurrent', res.experience);
          let result = new ExpResult();
          result.expGained = exp;
          result.oldLevel = oldLevel;
          result.newLevel = newLevel;
          let currentUser = GlobalContext.getContext().getObject('currentUser') as FriendInfo;
          let queryUsername = currentUser ? currentUser.username : '';
          FriendInfoApi.queryByUsername(queryUsername, (me: FriendInfo | null) => {
            if (me) {
              me.experience = res.experience;
              FriendInfoApi.updateData(me, () => { resolve(result); });
            } else {
              resolve(result);
            }
          });
        } else {
          reject(flag);
        }
      });
    });
  });
}
```

### 等级计算（HomeViewModel.ets）

```typescript
static getLevel(experience: number): number {
  return Math.floor(experience / HomeStore.EXP_PER_LEVEL) + 1;
}

static getLevelExp(experience: number): number {
  return experience % HomeStore.EXP_PER_LEVEL;
}
```

### 经验值动画+进度条（UserBaseInfo.ets）

@StorageLink('experienceNew') @Watch('onExpNewChange') experienceNew: number = 0;

onExpNewChange() {
  if (this.experienceNew > 0 && this.lastAnimatedNew !== this.experienceNew) {
    this.lastAnimatedNew = this.experienceNew;
    this.startExpAnimation(this.experienceOld, this.experienceNew);
  }
}

startExpAnimation(oldExp: number, newExp: number) {
  let current = oldExp;
  let timer = setInterval(() => {
    current += 1;
    if (current >= newExp) {
      current = newExp;
      clearInterval(timer);
    }
    this.displayLevel = HomeStore.getLevel(current);
    this.displayExp = HomeStore.getLevelExp(current);
  }, 20);
}

// 进度条 UI
Row() {

### 打卡经验弹窗（ExpGainPopup.ets）

```typescript
@Component
export struct ExpGainPopup {
  @Prop expGained: number = 0;
  @State private displayExp: number = 0;

  build() {
    Row() {
      Text('+').fontSize(20).fontWeight(FontWeight.Bold).fontColor(Color.White)
      Text(`${this.displayExp}`).fontSize(24).fontWeight(FontWeight.Bold).fontColor(Color.White)
      Text(' EXP').fontSize(16).fontWeight(FontWeight.Bold).fontColor(Color.White).opacity(0.9)
    }
    .padding({ left: 24, right: 24, top: 12, bottom: 12 })
    .borderRadius(24)
    .backgroundColor('rgba(255,165,0,0.90)')
    .shadow({ radius: 12, color: 'rgba(255,165,0,0.4)', offsetY: 4 })
  }

  startCountUp() {
    let current = 0;
    let target = this.expGained;
    let step = Math.max(1, Math.floor(target / 15));
    let timer = setInterval(() => {
      current += step;
      if (current >= target) { current = target; clearInterval(timer); }
      this.displayExp = current;
    }, 40);
  }
}
```

### 升级弹窗（ExpGainPopup.ets）

```typescript
@Component
export struct LevelUpPopup {
  @Prop newLevel: number = 1;
  @State private scaleValue: number = 0.3;
  @State private glowScale: number = 1;
  @State private glowOpacity: number = 0;

  build() {
    Column() {
      Stack() {
        Circle().width(160).height(160).fill('rgba(255,215,0,0.15)')
          .scale({ x: this.glowScale, y: this.glowScale }).opacity(this.glowOpacity)
        Column() {
          Text('LEVEL UP!').fontSize(18).fontWeight(FontWeight.Bold).fontColor('#FFD700')
          Text(`LV.${this.newLevel}`).fontSize(42).fontWeight(FontWeight.Bold)
            .fontColor(Color.White).margin({ top: 4 })
        }
      }
    }
    .width('100%').height('100%')
    .backgroundColor('rgba(0,0,0,0.65)')
    .onAppear(() => {
      animateTo({ duration: 400, curve: Curve.FastOutSlowIn }, () => {
        this.scaleValue = 1;
        this.glowScale = 2.5;
        this.glowOpacity = 0.6;
      });
      setTimeout(() => {
        animateTo({ duration: 600, curve: Curve.EaseOut }, () => {
          this.glowScale = 3.5; this.glowOpacity = 0;
        });
      }, 500);
    })
  }
}
```
